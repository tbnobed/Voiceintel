How classification works
After transcription, the NLP engine matches the transcript against keyword patterns for each category. The first matching category wins; if none match, the voicemail is assigned to General Inquiry. Renaming or adding categories takes effect on the next voicemail processed — existing voicemails can be re-analyzed from their detail page.
