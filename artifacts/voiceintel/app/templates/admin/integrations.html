How It Works

Inbound: Your phone system emails voicemail MP3 attachments to SendGrid, which POSTs them to the webhook URL above. VoiceIntel transcribes and analyzes each one automatically.

Outbound: Automation triggers with Notify Admin or Send Email actions deliver rich HTML alerts via SendGrid.

Docker env vars override any setting saved here. See .env.example in the project root for the full list.
