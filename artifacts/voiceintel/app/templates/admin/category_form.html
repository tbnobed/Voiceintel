Built-in keyword patterns

The NLP engine has hardcoded keyword patterns for the default categories. If you create a new category, voicemails won't be auto-assigned to it until a matching keyword pattern is added to the NLP service.

To teach the engine about your new category, edit app/services/nlp_service.py and add a pattern under CATEGORY_PATTERNS.

Renaming an existing category immediately updates all voicemails already assigned to it.
